# myblog
Blog System with Spring Boot
## 使用的技术
- 基础 Spring Boot 2.1.7
- 数据缓存 Redis
- 日志 slf4j
- 实时通信 websocket
- ORM框架 Mybatis
- MVC Spring MVC
- markdown处理框架 com.atlassian.commonmark
- 分页 com.github.pagehelper
- 数据库连接池 druid
- 前段引擎 thymeleaf
